# Armor HUD

A minimalistic client-side mod for Minecraft 1.21.x that displays your armor durability on the HUD.

  - **Orange**: Low durability (< 25%)

### âš™ï¸ Configuration
You can configure the mod directly within the game:
1.  **Press 'H'** (default) to open the Settings Menu.
2.  **Or navigate via:** `Mods -> Armor HUD -> Config`.

**Settings:**
- **Enable Warning**: Toggle the warning overlay on/off.
- **Threshold**: Adjust the percentage at which the warning appears (0-100%).
